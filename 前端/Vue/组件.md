# 注意事项
1. data必须是一个函数，实现一个闭包的环境，每个组件都有一个独立的值
2. 模版内容必须是单个根元素
3. 组件模版内容可以是模版字符串，需要ES6支持
4. 组件命名方式：驼峰式和短横线的方式，在标签里面必须使用短横线
