# JSONP
## 实现原理
`<script>` 标签不受浏览器同源策略影响，通过src属性，请求跨域的数据接口，并通过函数调用的形式，接受跨域接口响应回来的数据
## 实现方法
```html
<script src="http://localhost/api/jsonp?callback=success&name=zs&age=20">
</script>

<script>
function success(data){
	console.log(data)
}
</script>
```
 ## 缺点
 不能称为ajax
